Resources

Campaigns
These are the coupons, rewards, and specials

Resource Description
GET /api/:version/locations/:id/campaigns
GET /api/:version/campaigns/:id

Customers
The people who use the mobile app

Resource Description
POST /api/:version/customers/sign_in
POST /api/:version/customers/sign_out
POST /api/:version/customers/check_in
POST /api/:version/customers/check_out
POST /api/:version/customers/notification_token
GET /api/:version/customers/profile

Locations
The individual locations of a business

Resource Description
GET /api/:version/beacons/:uuid/location
GET /api/:version/locations/:id
GET /api/:version/locations/near/:lat/:lon
GET /api/:version/locations/map/:lat/:lon/:distance

Receipts
Receipts are associated with redemptions and are uploaded by the customer for approval

Resource Description
POST /api/:version/receipts
GET /api/:version/receipts/:status

Redemptions
Redemptions are what the customer created upon every redeemed campaign.

Resource Description
GET /api/:version/redemptions
POST /api/:version/redemptions

Share links
When a customer shares a campaign, a share link is created.

Resource Description
GET /api/:version/share_links/:id
POST /api/:version/share_links
