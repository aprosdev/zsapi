Receipts

POST /api/:version/receipts

Create a new receipt with the given params

Errors

422 Invalid receipt params were given (or invalid image).

Examples

200 OK

Params

Param name Description
receipt
required
Value:

Must be a Hash
receipt[redemption_id]
required

The associated redemption_id


Value:

Must be a number.

required

The location_id of where the redempetion was made (MAY REMOVE THIS)


Value:

Must be a number.
receipt[image]
required

The receipt image


Value:

Must be File.
