POST /api/:version/customers/sign_in

Signs the customer in by updating the social_token and social_type

Examples

200 OK

Params

Param name Description
customer
required
Value:

Must be a Hash
customer[email]
required

Email address of customer


Value:

Must be String
customer[social_token]
required

The social token returns by Facebook/Google+


Value:

Must be String
customer[social_type]
required

The type of social account connected


Value:

Must be one of: facebook, google+.
customer[social_id]
required

The social ID


Value:

Must be String
customer[first_name]
required

Customer's first name


Value:

Must be String
customer[last_name]
required

Customer's last name


Value:

Must be String
customer[social_friends]
optional

An array of the customer's social friends. Example: [1,4,5,6]


Value:

Must be an array of any type

POST /api/:version/customers/check_in

Checks in the customer to a location

Examples

{
  "response" => {
    "welcome_message", 		# The welcome greeting
    "points_earned", 		# The amount of points earned for checking in
    "special_campaign" => { 	# The exit campaign from their prior checkout (if available)
      #Campaign Meta
    }
  }
}

Params

Param name Description
location_id
required

The ID of the location


Value:

Must be a number.

POST /api/:version/customers/check_out

Checks OUT the customer FROM a location and returns the exit message

Examples

{
  "response" => {
    "exit_message", 		# The exit greeting
    "wait_time", 		# The amount time until the special campaign expires
    "special_campaign" => { 	# The exit campaign that can be redeemed on their next check in
      #Campaign Meta
    }
  }
}
